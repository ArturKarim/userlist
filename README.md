Тестовое задание:

Веб-приложение для отображения и фильтрации списка пользователей.

Технологический стэк:
Vue3, TS, Pinia.

Реализован список пользователей с Именем и Фамилией, Возрастом и Э. почтовым ящиком.
Добавлена возможно загрузки фото пользователей и отображения рядом с Именем и Фамилией.
В стилях реализована адаптивная верстка.
Дополнительно создана возможность фильтровать пользователей по возрасту, а так же сортировка по имени и возрасту.

Веб-приложение загружено в репозитории GitHub Pages
